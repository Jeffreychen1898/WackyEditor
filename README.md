# Wacky Editor
A text editor project intended for me to
learn and explore the different components
and technologies that goes into making a
text editor.

## How to use
Install ncurses
 - MAC: `brew install ncurses`

Compile the code
 - `make all`

Run the text editor
 - `./Wacky`
